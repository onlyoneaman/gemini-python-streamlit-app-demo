### gemini-python-streamlit-app-demo

Simple Chatbot using gemini, python and streamlit

### Start Virtual Environment

```bash
source venv/bin/activate
```

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
streamlit run app.py
```
